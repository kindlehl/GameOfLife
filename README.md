# GameOfLife

This is a rendition of the famous: Conway's Game of Life!

## __Dependencies__  
* SFML 2.4  

## __Build__

To build, simply run:  
  
`$ make`  

The executable will be in the current working directory, named GameOfLife.  

## __Controls__  

* Left-Click to toggle a cell's state
* Space to pause/resume 'game'
